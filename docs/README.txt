Upload your customer-facing PDFs into this folder using these EXACT filenames
(the links on the website already point to them):

  schematic.pdf     - full board schematic (export from EasyEDA Pro)
  user-guide.pdf    - setup, features & troubleshooting guide
  pinout.pdf        - printable pin reference diagram

How to upload on GitHub: open this "docs" folder in your repository,
click "Add file" -> "Upload files", drag the PDFs in, then "Commit changes".
The site republishes automatically in about a minute.

You can add more files here later (e.g. 3d-model.step, dimensions.pdf) -
just add a matching row to the Downloads table in index.html.
