Images for the nRF5340 BLE Dev Board page. Upload with these EXACT names:

  top_view_V1.jpg     - front render/photo  (hero + board tour + the hub card)
  bottom_view_V1.jpg  - back render/photo   (board tour)
  pinout.png          - page 1 of docs/pinout.pdf, rendered as PNG

Photo versions are controlled by IMG_VERSION at the top of ../index.html
(currently "V1"). To post new photos later: save them as ..._V2.jpg, set
IMG_VERSION = "V2", and update the card image path in the root index.html.

Tip from the ESP32 board: renders on a pure-black background make the board
appear to float in the dark hero section.

(This file just keeps the folder in git and documents the names - safe to
delete once the images are uploaded.)
