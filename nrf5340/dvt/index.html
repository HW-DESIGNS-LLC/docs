Findings so far

Caught before a single board was built

DVT starts before the soldering iron warms up. Reconciling the fabrication data against the schematic ahead of the build turned up two items — both closed.

Two I²C pull-up resistors missing from the assembly BOM resolved pre-build

The layout and placement data include both pull-up resistors for the Qwiic / I²C bus, but they were absent from the parts list used for kitting — built as-listed, the I²C port would not have worked. Caught in the pre-build review; the parts are kitted for the first articles and the BOM is corrected for the production run. The bench test I2C-01 confirms them on every board.

Qwiic power-rail assignment verified fixed

An earlier design revision powered the Qwiic connector from the nRF5340’s measured rail, which would have polluted the board’s current-measurement feature with sensor draw. The 2026-07-04 fabrication netlist confirms the connector now sits on the main 3.3 V rail — plugged-in sensors no longer appear in your µA measurements. Test IMEAS-09 re-verifies this on hardware.
